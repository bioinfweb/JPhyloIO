XML metadata demo application for JPhyloIO
==========================================

This example application demonstrates reading and writing XML metadata with 
JPhyloIO. An overview on this example is provided at 
<http://r.bioinfweb.info/JPhyloIODemoXMLMetadata>.

In order to run this demo application in your own Eclipse Workspace you need 
to checkout additional dependency projects from the following SVN locations:

- <https://secure.bioinfweb.info/Code/svn/commons.java/trunk/main/info.bioinfweb.commons.core>
- <https://secure.bioinfweb.info/Code/svn/commons.java/trunk/main/info.bioinfweb.commons.bio>
- <https://secure.bioinfweb.info/Code/svn/JPhyloIO/trunk/main>
